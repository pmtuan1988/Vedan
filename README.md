# Vedan - GitHub Pages

Trang quy trinh ban hang da duoc dua len nhanh `main` voi file `index.html`.

## Cach bat xem truc tiep

1. Vao `Settings` -> `Pages` trong repo.
2. Tai `Build and deployment`:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
3. Bam `Save`.

Sau 1-3 phut, mo link:

`https://pmtuan1988.github.io/Vedan/`
